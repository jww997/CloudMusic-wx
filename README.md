# CloudMusic-wx

仿网易云音乐的微信小程序 需下载[node服务器](https://github.com/Binaryify/NeteaseCloudMusicApi), 并启动

[API文档](https://binaryify.github.io/NeteaseCloudMusicApi/#/)来自[@Binaryify](https://github.com/Binaryify), 非常感谢! 👍 👍

### 😎 已实现

* 搜索
* 歌单
* 歌词
* 专辑
* 视频MV
* 排行榜
* 播放条

### 👏 待开发

* 登录
* 歌手
* 歌单广场
* 电台
* 私人FM
* 更多新碟
* 新歌推荐
* 每日推荐
* 音质切换
