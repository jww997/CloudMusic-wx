# CloudMusic-wx

# 仿网易云音乐的微信小程序
# 需下载[node服务器](https://github.com/Binaryify/NeteaseCloudMusicApi),并启动node app.js

### 已实现

* 专辑
* 歌词
* 歌单
* 视频MV
* 搜索
* 排行榜
* 播放条

### 未实现
