# CloudMusic-wx

仿网易云音乐的微信小程序

需下载[node服务器](https://github.com/Binaryify/NeteaseCloudMusicApi), 并启动node app.js 🤣 😘

[API文档](https://binaryify.github.io/NeteaseCloudMusicApi/#/)来自[@Binaryify](https://github.com/Binaryify), 非常感谢! 👍 👍

### 😎 已实现

* 专辑
* 歌词
* 歌单
* 视频MV
* 搜索
* 排行榜
* 播放条
